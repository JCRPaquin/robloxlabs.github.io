ROBLOX on GitHub

A GitHub organization to collaborate on ROBLOX projects and publish them

The ROBLOX on GitHub organization is used to host and group together projects related to ROBLOX and allow developers to collaborate on them. This makes it easier to find projects related to ROBLOX on GitHub and makes it easier for developers to collaborate on repositories, for repositories that are meant for collaboration. You can request to become a member of the organization or to get a repository hosted on this organization by asking on the issue manager of the organization's website's repository or by contacting the user "Sauraus" on GitHub. There are no requirements for joining the organization or hosting a repository on it. The organization was created by Mark Otaris in 2013 and was later transferred to the ROBLOX Corporation, which now has ownership of it, but does not explicitly endorse it.

Links

The roblox.github.io repository is used as the main repository for the organization: its wiki is used for the whole organization, and its issue tracker can be used to discuss issues and suggestions for the organization.
