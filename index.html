ROBLOX on GitHub

A GitHub organization to collaborate on ROBLOX projects and publish them

The ROBLOX on GitHub organization is used to host and group together projects related to ROBLOX and allow developers to collaborate on them. This makes it easier to find projects related to ROBLOX on GitHub and makes it easier for developers to collaborate on repositories, for repositories that are meant for collaboration. You can request to become a member of the organization or to get a repository hosted on this organization by contacting its owner, Mark Otaris, or any collaborator. There are no requirements for joining the organization or hosting a repository on it. The organization is managed by community members and is not affiliated with or endorsed by ROBLOX Corporation.
