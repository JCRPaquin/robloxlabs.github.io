ROBLOX LABS on GitHub

A GitHub organization for builders and players to collaborate on ROBLOX projects and publish them

The ROBLOX LABS on GitHub organization is used to host and group together projects related to ROBLOX and allow builders and players to collaborate on them. <\br>This makes it easier to find projects related to ROBLOX on GitHub and makes it easier for builders and players to collaborate. You can request to become a member of the organization or to get a repository hosted on this organization by asking on the issue manager of the organization's website's repository or by contacting the users "mark-otaris" or "Sauraus" on GitHub. <\br>The original ROBLOX GitHub organization was created by Mark Otaris in 2013 and was later transferred to the ROBLOX Corporation, which now has ownership of it. but does not explicitly endorse it.

Links

The robloxlabs.github.io repository is used as the main repository for the organization: its wiki is used for the whole organization, and its issue tracker can be used to discuss issues and suggestions for the organization.
